Description!

This is a website styling framework that uses lots of transparency. It is designed to look modern and easy to use.

These styles are free to use under the GPL licence.

More information can be found below, in the LICENCE file, and in the Readme.md file.

Read the source HTML code for examples of how to use the stylesheets.

This description box is recommended to be smaller than the screen size and is intended for breif descriptions. It can be made smaller, larger, or even used repeatedly!

Be careful to use sizes correctly so that the boxes are filled in a way that feels correct to the end user!

More information here!

This is where you put more detailed information.

This stylesheet set has multiple "Themes", which each look different. Currently, only the "Forest" theme is supported, but custom themes can be made by editing the colors in the stylesheets.

Use this theme by linking the stylesheets into your own HTML and using the CSS classes defined here. Use the HTML files here as examples to guide your own site!

An important thing to keep in mind, is that the blur effect is not currently supported on Firefox (97, at time of writing). This may change in the future, but keep in mind when creating your own themes that the site needs to look good even without the blur effect.
